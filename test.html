Icon Information

This extension uses a single icon (icon-light.png) in the toolbar.

Firefox does not currently support automatic theme-aware icon switching through the manifest.

For best results in both light and dark themes, use an icon with a transparent background or one that contrasts well with both light and dark backgrounds.
